Project Overview

This project replicates a modern headphone product landing page. The layout, typography, and spacing are inspired directly from the Figma design.

The page is fully responsive and provides smooth hover/active interactions for links and buttons, ensuring a polished user experience.

Getting Started

Clone the repository:

git clone https://github.com/Qortana/alx_html_css.git


Navigate to the project folder:

cd alx_html_css/headphones


Open index.html in a browser to view the landing page.

Optional: Open the project in a code editor like VS Code for further development.
